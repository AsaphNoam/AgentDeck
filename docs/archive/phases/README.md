not instructions and not product authority. Start at [`docs/specs/README.md`](../../specs/README.md),
then use [`AGENT-WORKFLOW.md`](../../features/AGENT-WORKFLOW.md) and the live
[`HANDOFF.md`](../../features/HANDOFF.md).

maintained. Git history is the authority for their original context.
